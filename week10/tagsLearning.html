< html >


This element can tell the browser that it itself is an HTML document.


< head >


The < head > tag is used to define the head of a document, which is a container for all the head elements. The elements in the < head > can refer to the script, indicate where the browser can find the stylesheet, provide meta information, and so on. The head of the document describes the various properties and information of the document, including the title of the document, the location in the Web, and the relationship with other documents. Most of the data contained in the head of a document will not really be displayed as a content to the reader. The following tags can be used in the head parts: < base >, < link >,< meta > ,< script > ,< style > , and < title >. < title > defines the title of a document, which is the only essential element in the head section.


< body >


The body element defines the body of the document. The body element contains all the contents of the document .such as text, hyperlinks, images, tables, lists, and so on.


Some important tags

< a >

The < a > tag defines a hyperlink, which is used to link from one page to another.

The most important attribute of the < a > element is the href attribute, which indicates the target of the link.

EXAMPLE:< a href="hellohtml.html" >hellohtml.html< /a >

< img >

The IMG element embeds an image into the web page.

Please note that, technically, the < img > tag does not insert the image in the web page,

but links the image from the web page. The < img > tag creates space for the occupying space of the referenced image.

The < img > tag has two essential attributes: the SRC property and the alt attribute.

EXAMPLE:< img img src="exampleJPG.jpg" alt="exampleJPG" / >exampleJPG< /a >
